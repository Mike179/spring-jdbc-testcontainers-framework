# Enterprise Spring JDBC Testcontainers Framework

Features:
- Create / Update / Delete User
- Transfer Money
- Rollback Transaction
- Idempotent Payment
- Concurrent Requests
- PostgreSQL Testcontainers
- Allure
- Jenkins Pipeline
- Jenkins Credentials support

Interview topics covered:
ACID, Commit, Rollback, Isolation, Idempotency, Race Conditions.
